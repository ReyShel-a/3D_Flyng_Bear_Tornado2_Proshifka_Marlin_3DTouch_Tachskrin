# 3D_Flyng_Bear_Tornado2_Proshifka_Marlin_3DTouch_Tachskrin
Прошивка для 3д Принтера Flyng Bear Tornado2 с тачскрином и датчиком автоуровня 3DTouch

----------------------------------------------------------
Этапы.
Вытащить тачпад из шлейфа CAN, подключить USB и БП
Компилировать прошивку и зашить в принтер с помощбю ПО VS Code и расширения PlatformIO выбираем наш COM порт
Удалить старые записи EPROM с памяти и занести новые в память EPROM из прошивки
Включить подогрев и сопло
Откалибрвать по оси Z зазор с листком бумаги до 0.05 мм с BLTouch
Откалиброванные изменеиия внести в "#define NOZZLE_TO_PROBE_OFFSET { -43, -12, -5.6 }" (расположение датчика 3D Thouch от сопла, штатное крепление)

Построить карту стола и сохранить ее
Запустить выравнивание

----------------------------------------------------------
Команды для отправки команд через терминал на подключенный порт 3D Принтера.

## 1. Проверка базовых настроек (Сброс памяти)
После прошивки в памяти могли остаться старые "хвосты". Сделаем чистый запуск:

* M502 — загрузить заводские настройки из твоего нового кода.
* M500 — сохранить их в EEPROM.
* M501 — вывести текущие настройки. Проверь строку M851, там должен стоять твой Z-Offset (например, -5.60).

## 2. Проверка 3D Touch и концевиков

* M119 — все оси (X, Y, Z) должны быть open.
* M280 P0 S10 — игла должна высунуться.
* M280 P0 S90 — игла должна убраться.
* M280 P0 S160 — сброс ошибки (если датчик мигает красным).

## 3. Проверка «Дома» (G28)

1. Команды нагрева:
M140 S60 — Нагреть стол до 60°C (не ждет нагрева, можно вводить следующую команду).
M104 S200 — Нагреть сопло до 200°C (не ждет нагрева).
2. Команды ожидания (если хочешь, чтобы принтер сам сказал, когда готов):
M190 S60 — Нагреть стол и ждать, пока температура доползет до 60.
M109 S200 — Нагреть сопло и ждать, пока будет 200.


* G28 — принтер должен найти X, Y и выехать в центр стола для замера Z.
* Внимание: Если игла коснется стола, а сопло при этом будет слишком близко — сразу выключай питание!

## 4. Проверка зазора Z (Главный тест с листком)
Чтобы понять, правильно ли задан офсет:

   1. Сделай G28.
   2. Введите G1 Z0 F3000 — сопло опустится в «ноль».
   3. Положи лист бумаги под сопло.
   * Идеал: Лист должен проходить с легким сопротивлением («закусывать»).
      * Если сопло в воздухе: Твой Z-Offset слишком «маленький» (нужно больше в минус, например с -5.60 до -5.80).
      * Если сопло давит в стол: Офсет слишком «большой» (нужно ближе к нулю, например -5.40).
   
## 5. Как подправить зазор без перепрошивки:
Если лист не проходит или болтается, не шейся заново! Пиши в терминал:

* M851 Z-5.75 (установи новое число).
* M500 (сохрани).
* G28 (повтори парковку для проверки).
* M114 - текущие координаты

## 6. Проверка карты стола

* G29 — запусти построение карты (25 точек). Если принтер пройдет все точки и не выдаст Probing Failed — автоуровень работает!
* M420 V — посмотри на цифры кривизны стола.
